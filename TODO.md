# TODO / Commandes utiles

## Lancer le projet en local

```bash
npm install
npm run dev
```

## Verifier la qualite

```bash
npm run lint
npm run typecheck
npm test
npm run test:e2e
```

## Nettoyer les artefacts locaux

```bash
npm run clean
```

## Build production

```bash
npm run build
npm run start
```

## Docker

```bash
docker compose up --build
```
